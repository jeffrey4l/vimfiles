# Usage

vim

```
curl -L https://raw.githubusercontent.com/jeffrey4l/vimfiles/litter/vimrc -o ~/.vimrc
```

neovim

```
curl -L https://raw.githubusercontent.com/jeffrey4l/vimfiles/litter/vimrc -o ~/.config/nvim/init.vim
```
